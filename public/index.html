cotillion.band

website github

This project is a metal band website built with JavaScript and Sass without relying on libraries and frameworks.

The Intersection Observer API controls the animations and highlights the currently viewed section in the navbar. Sticky positioning mixes up fixed and scrolling elements while persisting the page flow. Embedded Bandcamp player allows visitors to enjoy the music without having to leave the site.
